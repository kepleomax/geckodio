# Геккодио — сайт студии low-code разработки

Маркетинговый сайт студии. Реализуется на чистом HTML / CSS / JavaScript без фреймворков.

## Структура проекта

```
Company/
├── index.html                  Главная страница
├── partials/
│   └── header.html             Хедер (подгружается на каждую страницу через fetch)
├── assets/
│   ├── css/
│   │   ├── reset.css           Нормализация стилей
│   │   ├── variables.css       Дизайн-токены (цвета, шрифты, отступы)
│   │   ├── base.css            Типографика, кнопки, @font-face
│   │   ├── header.css          Стили хедера
│   │   └── home-hero.css       Стили Hero-блока главной
│   ├── js/
│   │   ├── partials.js         Загрузка хедера/партиалов
│   │   ├── header.js           Поведение хедера (скролл, бургер)
│   │   └── hero-rotator.js     Анимация смены слов в Hero
│   ├── img/
│   │   └── logo.svg            Логотип Геккодио
│   └── fonts/                  (сюда кладутся файлы Suisse Intl)
└── README.md
```

## Как запустить локально

Сайт использует `fetch()` для подгрузки хедера, поэтому открытие `index.html` двойным кликом (протокол `file://`) не сработает. Нужен локальный HTTP-сервер.

**Самый простой способ — расширение Live Server в VS Code / Cursor:**

1. Установить расширение `Live Server` (Ritwick Dey).
2. Открыть `index.html`, нажать «Go Live» в правом нижнем углу.

**Альтернатива — Python:**

```powershell
python -m http.server 5500
```

Затем открыть [http://localhost:5500](http://localhost:5500) в браузере.

**Альтернатива — Node.js:**

```powershell
npx serve .
```

## Шрифт Suisse Intl

Сайт подключает Suisse Intl через `@font-face` (см. `assets/css/base.css`). Ожидаемые файлы в папке `assets/fonts/`:

- `SuisseIntl-Regular.woff2` и `.woff`
- `SuisseIntl-Medium.woff2` и `.woff`
- `SuisseIntl-Semibold.woff2` и `.woff`
- `SuisseIntl-Bold.woff2` и `.woff`

До тех пор пока файлы не добавлены, сайт автоматически использует **Inter** (подключён через Google Fonts) как визуально близкий аналог. После добавления файлов Suisse Intl подхватится без дополнительных правок.

Если имена файлов отличаются — скорректируйте пути в `assets/css/base.css` или переименуйте файлы в указанный выше формат.

## Фирменные цвета

| Цвет | HEX | Роль |
|---|---|---|
| Белый | `#FFFFFF` | Основной фон |
| Графит | `#1E1E1E` | Основной текст |
| Пурпурный | `#6B2EE5` | Primary — CTA, логотип, акценты |
| Салатовый | `#AAFF3F` | Подсветка слов в Hero, highlights |
| Жёлтый | `#FFFF00` | Точечные акценты, стикеры |
| Коралловый | `#EB5143` | Ошибки формы, дополнительные акценты |

Все цвета и другие токены вынесены в `assets/css/variables.css`.
